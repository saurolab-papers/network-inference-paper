There are a few additional steady states but this author is happy with this.  She explains: "this difference is caused because we used an old version of BoolNet for the paper. This old version did and exhaustive search, so we had to limit the extrinsic cytokines we tried. The result is that we did not try all possible combinations. Using the newest version of Boolnet I get a table like the one from GinSim."
